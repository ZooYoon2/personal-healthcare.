# personal-healthcare.

### 1.개인 건강관리 웹 사이트
 - 개인 건강관리에 필요한 기능을 제공하며 피트니스 정보를 제공, 추천하여 보다 사용자가 편리하게 개인 건강관리를 할수있게 돕는다.
 
### 2.역할
 - PL, 풀스택 개발
